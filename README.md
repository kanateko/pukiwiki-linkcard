# PukiWiki Linkcard Plugin

外部リンクから OGP（Open Graph Protocol）情報を非同期で自動取得し、見やすいリンクカードとして表示する PukiWiki 1.5.4 用のプラグインです。

## 特徴

- **非同期取得**: ページ読み込み時に即座にスケルトンUI（骨組み）を表示し、バックグラウンドで OGP 情報を取得・更新します。
- **キャッシュ管理**: `CACHE_DIR/linkcard/` に OGP 情報と画像を 7 日間（デフォルト）保存し、再取得の負荷を最小限に抑えます。
- **最適化された画像**: 取得した画像はサーバー側で適切なサイズにリサイズされ、WebP 形式に変換されるため、高速な表示が可能です。
- **レスポンシブデザイン**: モバイル端末からデスクトップまで、適切なレイアウト（テキスト左＋画像右）で表示されます。
- **ダークモード対応**: `PKWK_SKIN_DARK_THEME` 定数に基づいて自動的にテーマを切り替えます。
- **POST リクエスト**: 外部情報の取得リクエストには POST メソッドを使用し、セキュリティと信頼性を向上させています。

## 使い方

### ブロック型

```text
#linkcard(URL)
```

**例:**
```text
#linkcard(https://example.com/article)
```

## アーキテクチャ

1.  **PHP (convert)**: ページ内で `#linkcard(URL)` が呼び出されると、URL を属性に持つプレースホルダ（スケルトンUI）と、初回のみ必要なスクリプト/スタイルを出力します。
2.  **JavaScript**: ページの読み込み完了後に、各プレースホルダに対応するリクエストを `POST ?plugin=linkcard` に対して送信します。
3.  **PHP (action)**: リクエストを受け取り、キャッシュの確認、外部サイトからの OGP 取得、画像の保存・変換処理を行い、結果を JSON 形式で返します。
4.  **JavaScript**: JSON の結果に基づいて DOM を構築し、リンクカードを表示します。取得に失敗した場合は、自動的にシンプルなリンク形式にフォールバックします。
