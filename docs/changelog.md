# Changelog

## [1.1.0] - 2026-03-12

### セキュリティ

- SSRF (Server-Side Request Forgery) 対策を実装。プライベートIPアドレス（ローカルネットワーク等）へのリクエストを制限。
- 最大ダウンロードサイズ制限を追加（HTML: 2MB, 画像: 10MB）。

### 改善

- キャッシュの自動クリーンアップ（ガベージコレクション）を実装。30日を過ぎた古いキャッシュファイルを自動的に削除。
- タイムアウトの設定を cURL リクエストに明示的に適用。
- User-Agent (UA) をモダンなブラウザ（Chrome）と同等の文字列に変更し、取得成功率を向上。

## [1.0.2] - 2026-03-12

### セキュリティ

- CSRFトークンによるPOST認証を追加。外部サイトからの直接的なPOSTリクエストを制限し、セキュリティを強化。

## [1.0.1] - 2026-03-12

### 修正

- カードにホバーした際の外枠（アウトライン）のエフェクトを、より自然でモダンなスタイルに調整。

## [1.0.0] - 2026-03-12

### 追加

- 外部リンクから OGP（Open Graph Protocol）情報を非同期で取得してリンクカードを表示する機能を実装。
- ブロック型プラグインとして提供（`#linkcard(URL)`）。
- キャッシュ機能を実装。`CACHE_DIR/linkcard/` に OGP 情報（JSON）と画像を 7 日間（`PLUGIN_LINKCARD_CACHE_EXPIRE = 604800` 秒）保存。
- 取得した画像データのダウンロード、適切なサイズ（240x126px）へのリサイズ、および WebP 形式への変換（品質 80）を実装。
- フロントエンドでの非同期取得に、セキュリティと信頼性を考慮して POST リクエストを採用。
- スケルトンUI（Shimmer アニメーション付き）を導入し、読み込み中の視覚フィードバックを改善。
- ダークモード（`PKWK_SKIN_DARK_THEME` に連動）およびレスポンシブデザインに対応。
- フォールバック表示（OGP が取得できない場合やエラー発生時にシンプルなリンクを表示）を実装。
